# bluez-alsa for OpenWrt

OpenWrt package for **bluez-alsa**, a lightweight Bluetooth audio backend that integrates BlueZ with ALSA. It provides **A2DP** audio support and exposes Bluetooth audio devices as standard ALSA PCM interfaces.

This repository contains the OpenWrt Makefile, utilities, and required D-Bus/ALSA configuration files to run bluez-alsa on embedded systems.
